# Exemples :

## Creation d'une machine virtuelle pour Talos :
```hcl
module "proxmox_vm" {
  source = "../.."
  network_config = {
    gateway : "192.168.10.1"
    domain : "example.com"
    nameservers : [
      "192.168.10.100",
      "192.168.10.200",
      "1.1.1.1"
    ]
  }
  tags = ["talos-linux", "kubernetes"]
  vm_id  = 100
  vm_ip_address = "192.168.10.2"
  vm_name = "test-vm"
}
```
